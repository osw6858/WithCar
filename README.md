# WithCar
WithCar은 자바 스프링(Spring)과 MySQL을 활용한 MVC 패턴을 기반으로 한 자동차 공유 페이지입니다. 이 페이지는 사용자들이 자동차를 대여하거나 자신의 자동차를 공유할 수 있는 기능을 제공하여 편리하고 경제적인 자동차 이용을 가능하게 합니다.

## 주요 기능
- 사용자 등록 및 로그인
- 자동차 등록 및 대여
- 실시간 예약과 결제 시스템
- 리뷰 및 평점 기능
